# NerdCalculator
The application for playing with different number systems includes things like converting a number to different systems, as well as, among others, representing a number in different styles (little and big endian), bit shifting and adding numbers from different systems

## System Requirements

[Basic Requirements]

- C++ compilator Version 15.1.1 or later
- Ninja Version 1.12.1 or later
- CMake Version 3.20 or later
- Git Version 2.50.1 or later

[Optional Requirements]

- Clang-format Version 20.1.7 or later
